# Тестовое задание для компании "ИнфоТеКС"

## Запуск проекта

1. Установите зависимости:

```bash
npm install
```

2. Запустите приложение в dev-режиме:

```bash
npm run start
```

После этого приложение будет доступно на `http://localhost:8080`.

## Общая информация

- Проект **"Базы пользователей"** реализован в соответствии с предложенным стеком технологий.  
- Разработка велась в публичном github-репозитории, каждый этап разработки проходил в отдельной ветке.  
- После окончания работы над очередным этапом разработки, производился мердж результата в `master` ветку.

## Архитектура и структура

Проект разрабатывался в соответствии с **FSD архитектурой**. Так как проект достаточно небольшой для такой сложной архитектуры, возможны спорные моменты с распределением логики по слоям.

Для наглядности, компонент `LoginForm` и его логика разрабатывались внутри страницы `LoginPage`, так как не используются нигде больше (**По принципу FSD - не усложняй без необходимости**).

При этом хуки `useLogin` и `useLogout` все же вынесены в `features`, так как они могут быть использованы и в других компонентах (**Кнопка выхода на странице `users`**).

С другой стороны, логика работы `users` и модальные окна вынесены в `features` по соображениям явного масштабирования работы с пользователями при дальнейшей разработке проекта.
